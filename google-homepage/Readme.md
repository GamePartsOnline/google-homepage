# Google Homepage

This project is a simple reproduction of the Google homepage.

## Technologies used

- HTML
- CSS
- Flexbox

## Project goal

The goal of this project is to practice basic HTML structure and CSS layout by recreating the visual appearance of Google's homepage.

## Features

- Top navigation bar
- Centered Google-style logo
- Search bar
- Action buttons
- Footer navigation
